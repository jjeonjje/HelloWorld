# 클래스 내용 직접 삽입

클래스에 기능이 너무 적을 떈
*그 클래스의 모든 기능으 ㄹ다른 클래스로 합쳐 넣고 원래의 클래스는 삭제하자.*

* [클래스 내용 직접 삽입](4.InlineClass.md)은 [클래스 추출](3.ExtractClass.md)과 반대다
* 클래스가 더 이상 제 역할을 수행하지 못하여 존재할 이유가 없을 때 실시한다.
* 클래스를 가장 많이 사용하는 다른 클래스를 하나 고른 후 합쳐야 한다.
